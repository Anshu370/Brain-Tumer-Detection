§ 7.1

ML Model Architecture — CNN Brain Tumor Classifier

Stage 1 — Input
MRI Image Input & Preprocessing
Raw MRI image (grayscale/RGB) → Resize to 224 × 224 pixels → Random Horizontal Flip · Random Rotation ±10° · Color Jitter (brightness, contrast) → ToTensor → Normalize (μ=[0.485, 0.456, 0.406], σ=[0.229, 0.224, 0.225])
Stage 2 — Feature Extraction (CNN Backbone)
5× Convolutional Blocks
Conv Block 1
Conv2d 3→32
BN · ReLU
MaxPool 2×2
Conv Block 2
Conv2d 32→64
BN · ReLU
MaxPool 2×2
Conv Block 3
Conv2d 64→128
BN · ReLU
MaxPool 2×2
Conv Block 4
Conv2d 128→256
BN · ReLU
MaxPool 2×2
Conv Block 5
Conv2d 256→512
BN · ReLU
MaxPool 2×2
Each block: Conv2d (3×3, padding=1, bias=False) → BatchNorm2d → ReLU → MaxPool2d(2,2) → AdaptiveAvgPool2d(1,1) → Feature Vector [512]
Stage 3 — Flatten
Global Feature Flatten
torch.flatten(x, 1) → Output shape: [batch_size, 512]
Stage 4 — Fully Connected Classifier
3-Layer MLP Head
FC Layer 1
Linear(512 → 4096)
ReLU
→
FC Layer 2
Linear(4096 → 1024)
ReLU
→
FC Layer 3
Linear(1024 → 4)
Logits
Optimizer: Adam (lr=0.0001, weight_decay=1e-5) · Scheduler: StepLR (step=5, γ=0.8) · Loss: CrossEntropyLoss · Epochs: 25 · Batch size: 32
Stage 5 — Output
Softmax Classification → Tumor Class
Glioma
Meningioma
No Tumor
Pituitary
argmax(softmax(logits)) → predicted class + confidence score (%) → stored in MRI Report · Evaluated with: Precision, Recall, F1-score, Confusion Matrix
